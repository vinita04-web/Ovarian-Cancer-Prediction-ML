# 🩺 Ovarian Cancer Prediction Using Machine Learning

## 📌 Project Overview

The Ovarian Cancer Prediction System is a Machine Learning-based healthcare application developed to assess ovarian cancer risk using patient biomarker data.

The system analyzes important medical features such as HE4, CA125, Age, Menopause Status, and other biomarkers to predict whether a patient belongs to a lower-risk or higher-risk category.

The application is built using Python, Streamlit, and Machine Learning techniques to provide an interactive healthcare dashboard with additional wellness guidance.

---

## 🎯 Objectives

* Predict ovarian cancer risk using Machine Learning.
* Assist in early risk assessment.

## 🧠 Machine Learning Model

**Algorithm Used:** Random Forest Classifier

### Why Random Forest?

* High accuracy
* Handles multiple features effectively
* Reduces overfitting
* Suitable for medical datasets

---

## 📂 Dataset Information

Dataset Name:

* Ovarian Cancer Dataset

Dataset Size:

* Total Records: 235
* Features: 48
* Target Variable: TYPE

Class Distribution:

* Benign Cases: 146
* Ovarian Cancer Cases: 89

Training-Test Split:

* Training Data: 80%
* Testing Data: 20%

## ▶️ Installation

Clone the repository:

```bash

```

Navigate to the project directory:

```bash

```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

